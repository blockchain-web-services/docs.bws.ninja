# Main API Methods

Running blockchain solutions is an asynchronous process: You send data to the blockchain and then wait for the network to validate it.&#x20;

{% hint style="warning" %}
BWS Unified API uses only two methods to call for solution operations: Call and Fetch.\
\
The CALL operation starts the blockchain request. FETCH returns the status of the call.
{% endhint %}
