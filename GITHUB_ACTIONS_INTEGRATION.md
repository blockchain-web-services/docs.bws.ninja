# GitHub Actions Integration - Blurbs Generation

**Date:** 2025-11-30
**Status:** ✅ Complete - Ready for Production

## Overview

Successfully integrated the marketing blurbs generation system into the existing GitHub Actions workflow. The workflow now automatically generates both product snapshots and AI-powered marketing blurbs on a weekly schedule or manual trigger.

## What Changed

### GitHub Actions Workflow Updated

**File:** `.github/workflows/update-product-media.yml`

**New Steps Added:**

1. **Deploy snapshots to GitBook** (Step 4)
   - Runs `deploy-to-gitbook.cjs` to organize captured snapshots
   - Creates GitBook-formatted pages for product media

2. **Generate marketing blurbs** (Step 5)
   - Runs `generate-blurbs.cjs --all` to generate blurbs for all products
   - Uses `ANTHROPIC_API_KEY` from GitHub Secrets
   - Includes graceful fallback if API key is not configured
   - Set to `continue-on-error: true` to not block the workflow

3. **Updated artifact path**
   - Changed from `media/` to `media-assets/` to include both snapshots and blurbs

4. **Enhanced summary output**
   - Reports snapshot counts
   - Reports blurb generation statistics
   - Shows number of products, pages, and data files

## GitHub Secret Configuration

### Required Secret

**Name:** `ANTHROPIC_API_KEY`
**Value:** Your Anthropic API key
**Access:** Repository secret
**Status:** ✅ Already configured by user

The workflow checks if this secret is available:
- **If present:** Generates blurbs for all products using Claude API
- **If missing:** Skips blurb generation with a warning message

## Error Handling

### Graceful Degradation

1. **Missing API Key**
   - Workflow continues without blurb generation
   - Warning message displayed in logs
   - Summary indicates blurbs were skipped

2. **API Failures**
   - Step marked with `continue-on-error: true`
   - Workflow completes even if blurbs fail
   - Partial results are committed if any succeed

3. **Rate Limiting**
   - 1-second delay between API calls built into script
   - Should handle 8 products × 7 channels = 56 calls without issues
   - If rate limited, will fail gracefully

## Rollback Plan

If issues arise, the workflow can be reverted:

### Disable Blurbs Generation

**Option 1: Remove API Key**
- Delete `ANTHROPIC_API_KEY` from GitHub Secrets
- Workflow will skip blurbs automatically

**Option 2: Comment Out Step**
```yaml
# - name: Generate marketing blurbs
#   id: generate_blurbs
#   ...
```

**Option 3: Revert Commit**
```bash
git revert <commit-hash>
git push
```

The workflow will continue to work for snapshots even if blurbs fail.

## Support & Troubleshooting

### Common Issues

**Issue: Blurbs not generated**
- Check ANTHROPIC_API_KEY is set in repository secrets
- Verify API key has sufficient credits
- Review workflow logs for error messages

**Issue: API rate limit errors**
- Normal for large runs, will retry on next schedule
- Can run manually with fewer products
- Check Anthropic account limits

**Issue: Invalid blurbs (wrong format)**
- Review GENERATION_REPORT.md for validation warnings
- Regenerate specific products if needed
- Adjust prompts in blurb-generator.cjs if persistent
